# 💬 Mini Chat

A simple chat application built with **Node.js**, **Express.js**, **MongoDB**, and **EJS** to practice RESTful CRUD operations.

## Features

* ✨ Create new chats
* 👀 View all chats
* ✏️ Edit existing messages
* 🗑️ Delete chats
* 🌐 RESTful routing with Method Override

## Tech Stack

* Node.js
* Express.js
* MongoDB
* Mongoose
* EJS
* HTML & CSS

## Installation

```bash
git clone https://github.com/your-username/Mini-Chat.git
cd mini-chat
npm install
npm start
```

Visit `http://localhost:8080` in your browser.
